186너는 AI 비서이자 멀티에이전트 코디네이터야.  

187사용자의 요청을 분석하여, 각 에이전트(Agent)와 도구(Tool)의 전문성을 최대한 활용해 최적의 결과를 만들어내.

189 

190[1. 에이전트/도구 역할 및 책임] 

191- 각 에이전트/도구는 자신의 전문 영역에만 집중하여, 책임 범위 내에서만 결과를 생성해야 해. 

192- 복합 요청의 경우, 각 에이전트가 순차적 또는 병렬적으로 협업하여 중간 결과를 공유하고, 최종 통합 결과를 생성해. 

193- 필요하다면, 중간 결과를 다른 에이전트에게 전달하여 추가 분석/가공/확장 작업을 수행해. 

194 

195[2. A2A 구조 및 협업 단계] 

196- (1) 요청 분석 → (2) 에이전트/도구 분배 → (3) 개별 실행 → (4) 중간 결과 취합 → (5) 통합/후처리 → (6) 최종 결과 생성 

197- 각 단계에서 수행한 작업, 사용한 도구, 중간 결과를 명확하게 기록(log)하고, 필요시 상세 근거와 출처를 남겨. 

198- 협업이 필요한 경우, 각 에이전트의 결과를 명확히 구분하여 통합하되, 중복/충돌/누락이 없도록 검증해.

202- 사용자의 요청 맥락(목적, 난이도, 톤, 길이, 포맷 등)을 파악하여, 맞춤형으로 결과물을 생성해. 

203- 복잡한 데이터/코드는 표, 코드블록, 시각화, 단계별 설명 등으로 명확하게 제시. 

204- 결과물의 한계나 불확실성이 있다면 반드시 명시하고, 추가 질문/확장 가능성도 안내해. 

205- 상세 답변(detailed_text)은 항상 JSON 형식으로 전달해야 하며, 표, 코드, 링크 등 다양한 포맷을 포함할 수 있다.

294 logging.info("[assistant_core] DataAnalysisTool preview generated (len=%d)", len(csv_preview)) 

295 elif result.get("text"): 

296 text_preview = result.get("text", "")[:2000] 

297 preview = f"\n문서 텍스트 요약(최대 2000자):\n{text_preview}"

299 else: 

300 preview = "\n(표/텍스트를 추출하지 못했습니다)" 

301 logging.warning("[assistant_core] DataAnalysisTool produced no tables/text for %s", file_path) 

302 except Exception: 

303 # 실패 시 엑셀 기본 프리뷰로 폴백 

304 if lower.endswith((".xlsx", ".xls")): 

305 try: 

306 import pandas as pd 

307 # 엔진 지정 시도 (xlsx=openpyxl, xls=xlrd) 

308 engine = None 

309 if lower.endswith(".xlsx"): 

310 engine = "openpyxl" 

311 elif lower.endswith(".xls"): 

312 engine = "xlrd" 

313 try: 

314 df = pd.read_excel(file_path, sheet_name=0, engine=engine) 

315 except Exception: 

316 # 엔진 미설치 등으로 실패 시 기본 동작 재시도 

317 df = pd.read_excel(file_path, sheet_name=0) 

318 df_preview = df.head(10) 

319 csv_preview = df_preview.to_csv(index=False) 

320 if len(csv_preview) > 2000: 

321 csv_preview = csv_preview[:2000] + "\n... (truncated)" 

322 preview = f"\n엑셀 프리뷰(첫 시트 상위 10행, CSV 형식):\n{csv_preview}" 

323 logging.info("[assistant_core] Fallback Excel preview generated (len=%d)", len(csv_preview)) 

324 except Exception: 

325 preview = "\n(엑셀/문서 프리뷰를 읽지 못했습니다. 필요 패키지 설치 여부를 확인하세요: pandas, openpyxl, (xls의 경우 xlrd))" 

326 logging.exception("[assistant_core] Excel fallback preview failed for %s", file_path) 

327 guidance = "\n지침: 업로드 파일 컨텍스트가 제공되었으므로, 이를 사용하여 분석/요약을 수행하세요. '파일 업로드 불가'라는 표현은 사용하지 마세요. 필요 시 도구를 호출해 표/데이터를 처리하세요. 제공된 프리뷰/표가 있다면 그것을 기반으로 바로 답변을 시작하세요." 

328 messages.append({"role": "system", "content": file_info + preview + guidance}) 

329 except Exception: 

330 pass 

331 

332 # 현재 사용자 입력 추가 

333 messages.append({"role": "user", "content": command_text}) 

334 

335 # --- LLM과의 대화 및 도구 사용 루프 --- 

336 while True: 

337 try: 

338 response = openai.chat.completions.create( 

339 model="gpt-4o", 

340 messages=messages, 

341 tools=loaded_tools_schemas, 

342 tool_choice="auto", 

343 ) 

344 response_message = response.choices[0].message

346 

347 # 1. LLM이 도구 사용을 결정한 경우 

348 if tool_calls: 

349 messages.append(response_message) # LLM의 도구 호출 결정도 대화 기록에 추가 

350 

351 for tool_call in tool_calls: 

352 function_name = tool_call.function.name 

353 function_to_call = loaded_tool_functions.get(function_name) 

354 function_args = json.loads(tool_call.function.arguments) 

355 

356 # ** A2A 핵심 로직: speak_text 도구 특별 처리 ** 

357 if function_name == "speak_text":

366

369 audio_bytes = function_to_call(**voice_args) 

370 

371 return { 

372 "status": "success", 

373 "response_type": "audio_response", # 새로운 응답 타입

375 "detailed_text": detailed_text, # UI에 표시될 상세 텍스트 

376 "audio_content": audio_bytes 

377 } 

378 

379 # 다른 일반 도구들 처리 

380 print(f"[Tool Call] Function: {function_name}, Args: {function_args}") 

381 function_response = function_to_call(**function_args) 

382 

383 # 바이너리 데이터 로깅 방지 

384 if isinstance(function_response, bytes): 

385 print(f"[Tool Response] {function_name} returned binary data of length: {len(function_response)} bytes") 

386 else: 

387 print(f"[Tool Response] {function_name} returned: {str(function_response)[:100]}{'...' if str(function_response) and len(str(function_response)) > 100 else ''}") 

388 

389 # 바이너리 데이터인 경우 안전한 방법으로 처리 

390 content_value = "[Binary data]" if isinstance(function_response, bytes) else str(function_response) 

391 

392 messages.append( 

393 { 

394 "tool_call_id": tool_call.id, 

395 "role": "tool", 

396 "name": function_name, 

397 "content": content_value, # 함수 결과는 문자열로 변환 (바이너리는 안전하게 처리) 

398 } 

399 ) 

400 # 도구 사용 결과를 바탕으로 LLM이 다시 생각하도록 루프 계속 

401 continue 

402 

403 # 2. LLM이 도구 사용 없이 직접 답변을 생성한 경우 (Fallback) 

404 # LLM이 speak_text를 사용하라는 지시를 어긴 경우에 해당 

405 final_response_text = response_message.content 

406 print(f"[Fallback Response] LLM generated text directly: {final_response_text}") 

407 return { 

408 "status": "success", 

409 "response_type": "text_fallback", 

410 "text_content": final_response_text, 

411 "audio_content": None # 오디오 없음 

412 } 

413 

414 except Exception as e: 

415 print(f"LLM 처리 중 오류 발생: {e}") 

416 return {"status": "error", "response": str(e)}
